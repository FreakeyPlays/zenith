## Run Locally

Clone the project

```bash
  git clone https://github.com/FreakeyPlays/zenith.git
```

Go to the project directory

```bash
  cd zenith/
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run start
```
